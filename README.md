Smart Plant Watering System | IoT Project

Project Overview
The Smart Plant Watering System is an IoT based automated irrigation solution designed to maintain proper soil moisture for plants. The system continuously monitors soil conditions using a soil moisture sensor and automatically activates a water pump when the soil becomes dry. This reduces manual monitoring and ensures efficient water usage.

Features

- Automatic watering based on soil moisture levels
- Real time soil moisture monitoring
- Relay controlled water pump activation
- Mobile notification and monitoring using the Blynk app
- Efficient and low cost smart irrigation system

Components Used

- ESP32 Microcontroller
- Soil Moisture Sensor
- Relay Module
- Water Pump
- Jumper Wires
- WiFi Network

Technologies Used

- Arduino IDE
- Embedded C Programming
- Internet of Things (IoT)
- Blynk Cloud Platform

Working Principle
The soil moisture sensor continuously measures the moisture level in the soil and sends the data to the ESP32 microcontroller. If the moisture level drops below a predefined threshold, the ESP32 activates the relay module to turn on the water pump and water the plant. When the soil reaches the required moisture level, the system automatically stops the pump.

Applications

- Smart home gardening
- Greenhouse monitoring systems
- Smart agriculture solutions
- Automated irrigation systems
